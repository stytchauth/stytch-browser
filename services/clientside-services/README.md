# One Tap Iframe

This package is an iframe RPC module used for communicating between the vanilla-js NPM module and various Stytch services
that are only available at runtime on the clientside.

For now:

- One Tap initialization and submit methods
- Method using awesome-phonenumber for phone input UI

It is hosted at

- [staging](https://js.staging.stytch.com/clientside-services/index.html)
- [prod](https://js.stytch.com/clientside-services/index.html)

### Setup:

```bash
cp .env.template .env.local
vi .env.local
```
