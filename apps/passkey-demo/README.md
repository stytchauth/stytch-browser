## React Demo

This is a demo app that uses the Passkeys Stytch SDKs in a NextJS environment.

Get started in a few seconds:

```bash
cp .env.template .env.local
vi .env.local
yarn start
```

Note: to use the above, you must run `yarn build` in the root of the repository first.

Alternatively, you can run `yarn run dev:passkey-demo` in the root of the repository.
