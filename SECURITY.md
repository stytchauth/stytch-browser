# Security Policy

## Reporting a Vulnerability

If you've found a security vulnerability, please follow our [responsible disclosure instructions](https://stytch.com/docs/resources/security-and-trust/security#:~:text=Responsible%20disclosure%20program).

Please include as much detail as possible to help us quickly identify and address the issue. We take all security reports seriously and will respond as soon as possible.

Do not disclose security issues publicly until they have been reviewed and addressed by the maintainers.
